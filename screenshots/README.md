# Screenshots

This folder contains screenshots of the Gym Training Generator app.

## How to add screenshots

1. Run the app on an emulator or physical device
2. Take screenshots using `adb shell screencap` or Android Studio's screenshot button
3. Save them here with descriptive names, e.g.:
   - `01_login_screen.png`
   - `02_register_screen.png`
   - `03_dashboard.png`
   - `04_workout_generator.png`
   - `05_exercise_library.png`
   - `06_workout_plan.png`
   - `07_progress_screen.png`
   - `08_history_screen.png`
   - `09_profile_screen.png`
   - `10_settings_screen.png`
   - `11_tablet_landscape.png`
   - `12_dark_mode.png`
