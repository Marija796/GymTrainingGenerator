# Videos / Screencasts

This folder contains screencast recordings demonstrating the Gym Training Generator app.

## Suggested screencast recordings

| File | Content |
|------|---------|
| `01_login_demo.mp4` | Login with Email, Google, Facebook, and Anonymous |
| `02_workout_generation.mp4` | Generating a personalized workout plan |
| `03_exercise_library.mp4` | Browsing and searching the exercise library |
| `04_progress_tracking.mp4` | Viewing workout history and progress stats |
| `05_profile_settings.mp4` | Updating profile and switching to Macedonian language |
| `06_dark_mode.mp4` | Dark mode and light mode toggle |
| `07_tablet_demo.mp4` | App running on a tablet in landscape mode |

## How to record

Use Android Studio's built-in screen recorder:  
**Run → Record Screen** or use `adb shell screenrecord /sdcard/demo.mp4`
